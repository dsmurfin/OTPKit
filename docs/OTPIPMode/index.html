Enumeration OTPIPMode 

public enum OTPIPMode

IP Mode

The Internet Protocol version used by a Component.

  • ipv4Only: Only use IPv4

  • ipv6Only: Only use IPv6

  • ipv4And6: Use IPv4 and IPv6
